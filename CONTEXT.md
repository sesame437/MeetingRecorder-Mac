# MeetingRecorder

MeetingRecorder is a personal meeting capture tool for local audio recording, live captions, and companion meeting notes.

## Language

**Recording**:
The saved meeting audio file produced by the app.
_Avoid_: Verbatim transcript, raw capture

**Live Captions**:
Short-lived local captions shown during a recording to help the user follow the meeting.
_Avoid_: Verbatim transcript, official transcript

**Meeting Notes**:
The companion Markdown document containing captions-derived transcript snippets and optional rolling summaries.
_Avoid_: Verbatim transcript, minutes

**Verbatim Transcript**:
A removed feature that produced a separate word-for-word transcript file for a one-off use case.
_Avoid_: Live captions, meeting notes
